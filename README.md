# 全视之眼·智能营销中枢 - 原型演示系统

## 项目简介

这是一个为"熵变智元"定制的智能营销中枢前端原型系统，完美融合了智能化理念与细粒度指标展示。系统采用深色模式科技风格，旨在为客户展示一个既能"看清数据"又能"智能决策"的现代化 BI 系统。

## 功能特性

### 🎯 核心功能模块

1. **全局指挥舱 (Dashboard)**
   - 大盘核心水位监控（5个关键KPI卡片）
   - 实时数据趋势展示
   - 异常预警提示

2. **智能素材复盘引擎**
   - 素材象限分布图（爆款/黑洞/潜力/待观察）
   - 视频深度指标分析（3秒完播率、50%进度、完播率、转化率）
   - 漏斗图展示用户流失分析

3. **多维归因钻取**
   - 渠道效能对比（巨量引擎 vs 腾讯广点通）
   - 人群包价值验证（中间库 vs 普通人群）
   - 多维度数据趋势分析

### 🎨 设计特色

- **深色模式科技风**：采用深色主题，体现数据智能感
- **响应式布局**：适配不同屏幕尺寸
- **交互式图表**：使用 Recharts 实现丰富的可视化效果
- **实时数据模拟**：完整的模拟数据支持演示需求

## 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **图表库**: Recharts
- **图标**: Lucide React
- **路由**: React Router

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:3000` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 数据说明

所有数据均为模拟数据，位于 `src/data/mockData.ts`，包括：

- **KPI数据**: 总花费、总线索量、线索成本、参会率、ROI
- **素材数据**: 8个素材的消耗和转化成本数据
- **视频指标**: Top 5 视频的深度分析数据
- **渠道数据**: 巨量引擎和腾讯广点通的对比数据
- **人群包数据**: 最近7天的中间库和普通人群对比数据

## 演示脚本建议

### 1. 开场：直击痛点
"各位领导，针对目前投放业务中存在的'看不清、算不准、动得慢'三大痛点，我们为您构建了这套'全视之眼'系统。"

### 2. 演示第一屏（指挥舱）
"请看大屏，我们打通了天择系统和CRM数据。以往您只能看到广告消耗，现在在这里，您可以直接看到'参会率'和'实际成交ROI'。如果线索成本波动超过20%，系统会自动亮红灯报警。"

### 3. 演示第二屏（素材引擎）
"这是我们最核心的智能引擎。系统会自动把素材划分为'爆款'、'黑洞'和'潜力'三类。比如这个位于红色区域的'黑洞素材'，消耗高但转化差，系统给出的建议是'立即关停'。"

### 4. 演示第三屏（多维报表）
"针对咱们关心的'中间库'人群价值，我们可以通过这个报表清晰地看到，虽然他们前端成本略高，但后端成交率是普通人群的3倍，数据证明了我们的人群包策略是有效的。"

## 注意事项

- 本项目为原型演示系统，所有数据均为模拟数据
- 不需要后端支持，所有功能均为前端展示
- 图表数据会根据模拟数据自动更新
- 支持时间筛选和项目筛选（UI已实现，数据筛选逻辑可后续对接）
